About The Project

Sentinel AI is a Machine Learning based Network Intrusion Detection System that monitors network behavior and detects malicious activity in real time. It combines supervised and unsupervised models with a live monitoring dashboard.

🤖 ML Detection Engine
Random Forest and Isolation Forest models classify traffic as normal or attack.
⚡ Real-Time Alerts
Flask + SocketIO pushes live events instantly without page refresh.
📊 Explainable AI
SHAP shows which features influenced predictions.
🗂 Dataset
NSL-KDD dataset used for training and evaluation.
🌍 Live Visualization
Real-time global map shows attack & normal traffic.
🔐 Use Cases
SOC monitoring, anomaly detection, cyber threat analytics.
